# React Website 1

Project to learn the basics of React.

## Getting Started

Setting up your project locally.

### Dependencies

* React
* React Router Hash Link
* React ScrollSpy
* React Spring
* AOS

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/g-llima/React-Website1.git
   ```
2. Delete ***package-lock.json*** file 
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm start
   ```
